# <I18n de="Top Skills — Living Catalog" en="Top Skills — Living Catalog" />

<De>
Ein **lebendiger Katalog** der besten Claude-Code-Skills aus Anthropic-Releases,
Community und YouTube-Walkthroughs. Wird kontinuierlich erweitert — neue
Quellen werden via `researchedOn` und der Changelog-Folie nachvollziehbar
dokumentiert.

**Wartungs-Workflow:**
1. Neue Skill-Quelle (YouTube-Video, Blog-Post, GitHub-Repo) wird in den Workshop-Chat geworfen
2. Inhalt wird per `youtube-knowledge-extractor` / WebFetch ausgewertet
3. Eine neue `<SkillCard>` mit Name, Was, Quellen, Install, Trigger, Tags wird ergänzt
4. `researchedOn` wird im Frontmatter aktualisiert
5. Eintrag im Changelog (Slide 99.06) wird ergänzt

**Tier-System:**
- ★ **Must-have** — würden wir in jeder Demo-Setup empfehlen
- ✓ **Useful** — situativ stark, lohnt sich anzuschauen
- ⚗ **Experimental** — frisch, vielversprechend, noch nicht produktionsbewährt
</De>

<En>
A **living catalog** of the best Claude Code skills from Anthropic releases,
community, and YouTube walkthroughs. Updated continuously — new sources are
tracked via `researchedOn` and the changelog slide.

**Maintenance workflow:**
1. New skill source (YouTube video, blog post, GitHub repo) dropped into chat
2. Content is parsed via `youtube-knowledge-extractor` / WebFetch
3. A new `<SkillCard>` with name, what, sources, install, trigger, tags is added
4. `researchedOn` in frontmatter is bumped
5. Changelog entry (slide 99.06) added

**Tier system:**
- ★ **Must-have** — we'd recommend in every demo setup
- ✓ **Useful** — situational, worth a look
- ⚗ **Experimental** — fresh, promising, not yet production-proven
</En>
