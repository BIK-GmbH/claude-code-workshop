# <I18n de="Wie ihr Schwung haltet" en="How to Keep Momentum" />

<De>
## 14-Tage-Plan

### Woche 1 — Setup absichern

| Tag | Aktion |
|---|---|
| Mo | Eure CLAUDE.md mit dem Team teilen + reviewen lassen |
| Di | Einen Skill commit-en, den ihr im Workshop gebaut habt |
| Mi | Erste Spec-Driven-Aufgabe in echter Sprint-Umgebung |
| Do | `/cost`-Tracking aktivieren, Budget-Cap setzen |
| Fr | Wochen-Review: was hat funktioniert, was nicht? |

### Woche 2 — Routine aufbauen

| Tag | Aktion |
|---|---|
| Mo | Zweiten Skill bauen (idealerweise team-shared) |
| Di | MCP-Server für eure wichtigste Daten-Quelle integrieren |
| Mi | Subagent für Code-Review konfigurieren + erstmals nutzen |
| Do | Spec → Plan → Code Cycle fürs zweite Feature |
| Fr | Team-Sync: Konventionen iterieren, Erkenntnisse teilen |

## Support-Kanäle (14 Tage inklusive)

- **Slack / Email-Support** — schnelle Fragen
- **30-Min Check-in** nach 14 Tagen — Status, neue Hürden, Folgeworkshop?

## Optional zubuchbar

- **Vertiefungs-Coaching** (1:1 oder Team) — €300/Stunde
- **Folgeworkshop „Advanced Augmented Working"** — 3–6 Monate später
- **Skill-/Agent-/CLAUDE.md-Bau** durch BIK — Aufwand nach Abstimmung

<NoteCard variant="tip" title="Erinnerung">
Der Workshop ist **kein End-Punkt**, sondern ein **Start**. Augmented Working wird Routine durch Wiederholung, nicht durch ein einzelnes Trainings-Event.
</NoteCard>

<En>
## 14-day plan

### Week 1 — secure the setup

| Day | Action |
|---|---|
| Mon | Share your CLAUDE.md with the team + have it reviewed |
| Tue | Commit one skill you built in the workshop |
| Wed | First spec-driven task in real sprint |
| Thu | Enable `/cost` tracking, set budget cap |
| Fri | Weekly review: what worked, what didn't? |

### Week 2 — build a routine

| Day | Action |
|---|---|
| Mon | Build a second skill (ideally team-shared) |
| Tue | Integrate MCP server for your most important data source |
| Wed | Configure subagent for code review + first use |
| Thu | Spec → plan → code cycle for the second feature |
| Fri | Team sync: iterate conventions, share findings |

## Support channels (14 days included)

- **Slack / email support** — quick questions
- **30-min check-in** after 14 days — status, new hurdles, follow-up workshop?

## Optional add-ons

- **Deep-dive coaching** (1:1 or team) — €300/hr
- **Follow-up "Advanced Augmented Working" workshop** — 3–6 months later
- **Skill / agent / CLAUDE.md building** by BIK — effort by agreement

<NoteCard variant="tip" title="Reminder">
The workshop is **not an end point** but a **start**. Augmented working becomes routine through repetition, not through a single training event.
</NoteCard>
