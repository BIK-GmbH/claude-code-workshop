# <I18n de="Retrospektive" en="Retrospective" />

<De>

**3-Spalten-Retro im Plenum:**

| Was war wertvoll? | Was war anstrengend? | Was probiere ich Montag aus? |
|---|---|---|
| (z.B. Spec-Driven funktioniert) | (z.B. Plan-Iteration fühlt sich langsam an) | (z.B. erste Skill für PR-Reviews) |
| | | |
| | | |

**Vorgehen:**
1. Jeder schreibt 5 Min für sich (Stickies oder digitales Board).
2. Reihum jeder ein Item pro Spalte vorlesen.
3. Cluster-Erkennung: was war für mehrere wertvoll / anstrengend?
4. Commitment-Runde: jeder benennt **einen konkreten Schritt** für die nächste Woche.

**Im Nachgang (von mir):**
- Foto / Export der Retro
- 2-Tages-Wrap-up-Mail mit Cheat-Sheet, Slides, Aufzeichnungen, Park-Liste-Antworten
- Kalender-Reminder nach 14 Tagen für Slack-Check-in

</De>

<En>

**3-column retro in plenary:**

| What was valuable? | What was hard? | What will I try on Monday? |
|---|---|---|
| (e.g. spec-driven works) | (e.g. plan iteration feels slow) | (e.g. first skill for PR reviews) |
| | | |
| | | |

**Process:**
1. Everyone writes for 5 min (stickies or digital board).
2. Round-robin: one item per column each.
3. Cluster: what was valuable / hard for several?
4. Commitment round: each names **one concrete step** for the next week.

**Follow-up (from me):**
- Photo / export of the retro
- 2-day wrap-up email with cheat sheet, slides, recordings, parking-lot answers
- Calendar reminder after 14 days for the Slack check-in

</En>
