# <I18n de="Agenda" en="Agenda" />

<De>
| # | Modul | Was ihr danach könnt |
|---|---|---|
| 1 | **Grundlagen & Mindset** | Augmented Working einordnen, Tool-Wahl begründen |
| 2 | **Setup & Basis-Konfig** | Claude Code lokal lauffähig + erste `CLAUDE.md` |
| 3 | **Skills, Subagents & MCP** | Eigenen Skill produktiv, MCP-Server angebunden |
| 4 | **Spec-Driven Development** | Spec → Plan → Code als wiederholbarer Workflow |
| 5 | **Best Practices, Anti-Patterns, Sicherheit** | Anti-Patterns erkennen, Compliance-Checkliste |
| 6 | **Hands-on: Eigene Projekte** | Echtes Ticket aus eurem Alltag ausgeliefert |

## Didaktischer Mix

- **60 % Anwendung / 40 % Theorie**
- **Live-Coding statt Folienschlachten** — jede Mechanik am offenen Editor
- **Hands-on-Übung nach jedem Modul** — geführt → frei
- **Begleitmaterial:** CLAUDE.md-Vorlagen, Skill-Beispiele, Cheat-Sheets (in diesem Deck unter Anhang)

<NoteCard variant="hint">
Pro Modul ca. 90 Min Theorie + 30 Min Übung. Pausen alle 90 Min.
</NoteCard>

<En>
| # | Module | Outcome |
|---|---|---|
| 1 | **Foundations & Mindset** | Position augmented working, justify your tool choice |
| 2 | **Setup & Configuration** | Claude Code running locally + first `CLAUDE.md` |
| 3 | **Skills, Subagents & MCP** | One own skill in production, MCP server wired in |
| 4 | **Spec-Driven Development** | Spec → plan → code as a repeatable workflow |
| 5 | **Best Practices, Anti-Patterns, Security** | Spot anti-patterns, apply compliance checklist |
| 6 | **Hands-on: Your projects** | Ship a real ticket from your daily work |

## Didactic mix

- **60 % practice / 40 % theory**
- **Live coding, not slide-shows** — every mechanic shown in the editor
- **Hands-on exercise after every module** — guided → free
- **Companion material:** CLAUDE.md templates, skill examples, cheat sheets (see appendix in this deck)

<NoteCard variant="hint">
~90 min theory + 30 min exercise per module. Breaks every 90 min.
</NoteCard>
