# <I18n de="Offene Fragen" en="Open Questions" />

<De>
## Format

- **15–20 Min** offene Q&A im Plenum
- Fragen aus dem Workshop **+** allgemeine Fragen
- Wenn keine Fragen kommen: meine vorbereiteten Themen

## Vorbereitete Themen (falls keine Fragen)

- **Cost-Management im Team** — Anthropic Console, pro-User-Budgets, Slack-Alerts
- **CI/CD-Integration** — Claude Code in GitHub Actions / GitLab CI
- **Multi-Tool-Workflows** — Claude Code + Cursor parallel sinnvoll einsetzen
- **Selbst-Hosting / Local Models** — wo OpenCode + Continue greifen
- **Anthropic-Roadmap** — was kommt 2026/2027
- **MCP-Ökosystem** — empfohlene Server, Eigenbau-Patterns

## Park-Liste

Fragen, die wir **nicht abschließend beantworten können**, gehen auf die Park-Liste:
- 1:1-Coaching-Termin (im 14-tägigen Slack-/Mail-Support)
- Folgeworkshop „Advanced Augmented Working" 3–6 Monate später
- BIK-Beratung für individuelle Kunden-Setups

<NoteCard variant="tip">
**Was-bringt-mir-das-morgen:** Stellt eine Frage so, dass die Antwort euch Montag früh konkret weiterbringt. Vermeidet „rein theoretisch …".
</NoteCard>
</De>

<En>
## Format

- **15–20 min** open Q&A in plenary
- Questions from the workshop **+** general questions
- If no questions: my prepared topics

## Prepared topics (in case of silence)

- **Team cost management** — Anthropic Console, per-user budgets, Slack alerts
- **CI/CD integration** — Claude Code in GitHub Actions / GitLab CI
- **Multi-tool workflows** — Claude Code + Cursor in parallel sensibly
- **Self-hosting / local models** — where OpenCode + Continue fit
- **Anthropic roadmap** — what's coming 2026/2027
- **MCP ecosystem** — recommended servers, build-your-own patterns

## Parking lot

Questions we **can't fully answer here** go on the parking lot:
- 1:1 coaching slot (in the 14-day Slack/mail support)
- Follow-up "Advanced Augmented Working" workshop 3–6 months later
- BIK consulting for client-specific setups

<NoteCard variant="tip">
**What-helps-me-monday:** phrase questions so the answer concretely helps Monday morning. Avoid "purely theoretically …".
</NoteCard>
</En>
