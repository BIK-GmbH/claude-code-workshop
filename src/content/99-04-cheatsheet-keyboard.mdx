# <I18n de="Tasten in diesem Workshop-Deck" en="Keys in This Workshop Deck" />

<De>
| Taste | Aktion |
|---|---|
| **→** / **j** / **Space** / **PageDown** | nächste Folie |
| **←** / **k** / **PageUp** | vorherige Folie |
| **Home** | erste Folie |
| **End** | letzte Folie |
| **⌘ + K** (Mac) / **Ctrl + K** | Command Palette / Suche |
| **P** | Presenter-Modus toggle |
| **F** | Vollbild toggle |

## Präsentations-Mode

| Aktion | Wie |
|---|---|
| **Aktivieren** | „▶ Präsentation"-Button im Header (oder `/p/<slide-id>`) |
| **Beenden** | `Esc` (oder „✕ Esc"-Button rechts oben) |
| **Speaker-Notes** | `N` toggelt overlay (oder Button im Top-Right) |
| **Theme** | `T` toggelt Light/Dark |
| **Vollbild** | `F` |

## URL-Parameter

| Parameter | Wirkung |
|---|---|
| `?presenter=1` | Speaker-Notes in Doku-Mode anzeigen |
| `/p/<slide-id>` | direkt im Präsentations-Mode starten |
| `/print` | Druckansicht (alle Folien linear) |

## Sidebar

- Klick auf Modul-Titel → Akkordeon auf/zu
- Klick auf Folien-Titel → direkt zur Folie
- Active-Folie ist hervorgehoben + linke Akzent-Border

## Mobile

- **Hamburger** oben links öffnet das Sidebar-Drawer
- **Swipe nach links** → nächste Folie (Slide gleitet mit Finger mit)
- **Swipe nach rechts** → vorherige Folie
- Vertikales Scrollen funktioniert weiterhin (vertikaler Swipe = Scroll)
- Im Präsentations-Modus genauso swipebar

## Animationen

- **Sparkles-Icon** im Header (✨) toggelt 3-stufig: **Auto → An → Aus → Auto**
- **Auto** (Default) respektiert das System-Setting `prefers-reduced-motion`
- **An**: Slide gleitet beim Wechsel von links/rechts ein
- **Aus**: alle Übergänge deaktiviert (für reduzierte Motion-Wahrnehmung oder schwache Geräte)
- Punkt rechts oben am Icon: grün = aktiv, grau = aus, kein Punkt = Auto-Modus

## Sprache & Theme

- Header oben rechts: **DE / EN** umschalten — wird gespeichert
- Sonne / Mond → Light/Dark — wird gespeichert
- System-Theme wird beim ersten Besuch übernommen

## PDF / Druck

- Browser-Print im `/print`-View → optimierte Druck-Ausgabe
- Sidebar/Header werden ausgeblendet
- Speaker-Notes werden mitgedruckt
</De>

## Presentation mode

| Action | How |
|---|---|
| **Enter** | "▶ Present" button in header (or `/p/<slide-id>`) |
| **Exit** | `Esc` (or "✕ Esc" top-right button) |
| **Speaker notes** | `N` toggles overlay |
| **Theme** | `T` toggles light/dark |
| **Fullscreen** | `F` |

## URL parameters

| Parameter | Effect |
|---|---|
| `?presenter=1` | show speaker notes in docs mode |
| `/p/<slide-id>` | start directly in presentation mode |
| `/print` | print view (all slides linearly) |

## Sidebar

- Click module title → accordion open/close
- Click slide title → jump directly
- Active slide is highlighted with a left accent border

## Mobile

- **Hamburger** top-left opens the sidebar drawer
- **Swipe left** → next slide (slide follows your finger live)
- **Swipe right** → previous slide
- Vertical scrolling still works (vertical swipe = scroll)
- Same swipe behaviour in presentation mode

## Animations

- **Sparkles icon** in the header (✨) cycles 3-state: **Auto → On → Off → Auto**
- **Auto** (default) respects the OS `prefers-reduced-motion` setting
- **On**: slides glide in from left/right on navigation
- **Off**: all transitions disabled (for reduced-motion preference or weak devices)
- Dot top-right of the icon: green = active, grey = off, no dot = auto mode

## Language & theme

- Top right header: **DE / EN** switch — persists
- Sun / moon → light/dark — persists
- System theme is adopted on first visit

## PDF / print

- Browser print on `/print` → optimised output
- Sidebar/header are hidden
- Speaker notes are printed
</En>
