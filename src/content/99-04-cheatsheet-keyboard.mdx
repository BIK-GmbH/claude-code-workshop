# <I18n de="Tasten in diesem Workshop-Deck" en="Keys in This Workshop Deck" />

<De>
| Taste | Aktion |
|---|---|
| **→** / **j** / **Space** / **PageDown** | nächste Folie |
| **←** / **k** / **PageUp** | vorherige Folie |
| **Home** | erste Folie |
| **End** | letzte Folie |
| **⌘ + K** (Mac) / **Ctrl + K** | Command Palette / Suche |
| **P** | Presenter-Modus toggle |
| **F** | Vollbild toggle |

## URL-Parameter

| Parameter | Wirkung |
|---|---|
| `?presenter=1` | Speaker-Notes anzeigen |
| `/print` | Druckansicht (alle Folien linear) |

## Sidebar

- Klick auf Modul-Titel → Akkordeon auf/zu
- Klick auf Folien-Titel → direkt zur Folie
- Active-Folie ist hervorgehoben + linke Akzent-Border

## Sprache & Theme

- Header oben rechts: **DE / EN** umschalten — wird gespeichert
- Sonne / Mond → Light/Dark — wird gespeichert
- System-Theme wird beim ersten Besuch übernommen

## PDF / Druck

- Browser-Print im `/print`-View → optimierte Druck-Ausgabe
- Sidebar/Header werden ausgeblendet
- Speaker-Notes werden mitgedruckt
</De>

<En>
| Key | Action |
|---|---|
| **→** / **j** / **Space** / **PageDown** | next slide |
| **←** / **k** / **PageUp** | previous slide |
| **Home** | first slide |
| **End** | last slide |
| **⌘ + K** (Mac) / **Ctrl + K** | command palette / search |
| **P** | toggle presenter mode |
| **F** | toggle fullscreen |

## URL parameters

| Parameter | Effect |
|---|---|
| `?presenter=1` | show speaker notes |
| `/print` | print view (all slides linearly) |

## Sidebar

- Click module title → accordion open/close
- Click slide title → jump directly
- Active slide is highlighted with a left accent border

## Language & theme

- Top right header: **DE / EN** switch — persists
- Sun / moon → light/dark — persists
- System theme is adopted on first visit

## PDF / print

- Browser print on `/print` → optimised output
- Sidebar/header are hidden
- Speaker notes are printed
</En>
