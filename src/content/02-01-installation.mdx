# <I18n de="Claude Code installieren" en="Install Claude Code" />

<De>
## Voraussetzungen

- **macOS 13+**, **Windows 10 1809+**, **Ubuntu 20.04+**, **Debian 10+** oder **Alpine 3.19+**
- **Terminal-Zugang** mit Bash, Zsh, PowerShell oder CMD
- **Git** empfohlen; auf nativem Windows **Git for Windows**, damit Claude das Bash-Tool nutzen kann
- **Node.js nur noch für npm-Fallbacks** nötig, nicht für den empfohlenen Native Installer

## Installation

<CommandBox lang="bash">
curl -fsSL https://claude.ai/install.sh | bash

# macOS Alternative:
brew install --cask claude-code

# npm nur falls ihr bewusst den Legacy/Fallback-Pfad nutzt:
npm install -g @anthropic-ai/claude-code
</CommandBox>

Auf Windows: PowerShell `irm https://claude.ai/install.ps1 | iex`, CMD `curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd`, oder `winget install Anthropic.ClaudeCode`.

## Erstes Login

<CommandBox lang="bash">
claude
</CommandBox>

Beim ersten Start öffnet sich der Auth-Flow im Browser. Login wahlweise via:

- **Claude Pro / Max** (Subscription) — empfohlen für Einzel-Devs
- **Anthropic API-Key** (Pay-per-use) — für Teams + CI/CD
- **Amazon Bedrock / Google Vertex** — für Enterprise-Compliance

## Quick-Sanity-Check

<CommandBox lang="bash">
claude --version
claude
# in der Session:
/doctor                # prüft Installation, Settings, MCP, Auth
</CommandBox>

<NoteCard variant="hint">
Windows wird nativ unterstützt. WSL2 bleibt sinnvoll, wenn eure Projekte Linux-Toolchains oder Sandbox-Isolation brauchen; für Windows-native Projekte reicht Git for Windows meist aus.
</NoteCard>
</De>

<En>
## Prerequisites

- **macOS 13+**, **Windows 10 1809+**, **Ubuntu 20.04+**, **Debian 10+**, or **Alpine 3.19+**
- **Terminal access** with Bash, Zsh, PowerShell, or CMD
- **Git** recommended; on native Windows use **Git for Windows** so Claude can use the Bash tool
- **Node.js is only needed for npm fallbacks**, not for the recommended native installer

## Install

<CommandBox lang="bash">
curl -fsSL https://claude.ai/install.sh | bash

# macOS alternative:
brew install --cask claude-code

# npm only if you intentionally use the legacy/fallback path:
npm install -g @anthropic-ai/claude-code
</CommandBox>

On Windows: PowerShell `irm https://claude.ai/install.ps1 | iex`, CMD `curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd`, or `winget install Anthropic.ClaudeCode`.

## First login

<CommandBox lang="bash">
claude
</CommandBox>

The first launch opens the auth flow in your browser. Sign in with:

- **Claude Pro / Max** (subscription) — recommended for individuals
- **Anthropic API key** (pay-per-use) — for teams + CI/CD
- **Amazon Bedrock / Google Vertex** — for enterprise compliance

## Quick sanity check

<CommandBox lang="bash">
claude --version
claude
# inside the session:
/doctor                # checks install, settings, MCP, auth
</CommandBox>

<NoteCard variant="hint">
Windows is supported natively. WSL2 is still useful for Linux toolchains or sandbox isolation; for Windows-native projects, Git for Windows is usually enough.
</NoteCard>
</En>
